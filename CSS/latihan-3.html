Latihan 3: Padding

Pendahuluan

Padding merupakan properti untuk memberikan ruang antara border elemen ke konten. Padding biasanya digunakan untuk desain yang menggunakan prinsip negative space untuk memberikan ruang kosong pada elemen. Format penulisan default properti padding menggunakan format padding: atas | kanan | bawah | kiri;
