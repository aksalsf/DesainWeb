NestedList1

Pendahuluan

Dengan menambahkan tag <ul> dalam tag <li> Anda bisa membuat sebuah sub-list yang bisa digunakan sebagai rincian dari suatu item.

Contoh koding:

<h3>Cara Atasi Monitor Mati</h3> <ul> <li>Periksa kabel daya monitor</li> <li>Periksa kabel VGA, bila longgar bisa dikencangkan</li> <li>Bersihkan RAM</li> <ul> <li>Cabut RAM dari slotnya</li> <li>Bersihkan kuningan RAM dengan penghapus karet</li> <li>Pasang kembali RAM</li> </ul> </ul>

Hasil:

Cara Atasi Monitor Mati

  • Periksa kabel daya monitor
  • Periksa kabel VGA, bila longgar bisa dikencangkan
  • Bersihkan RAM
    • Cabut RAM dari slotnya
    • Bersihkan kuningan RAM dengan penghapus karet
    • Pasang kembali RAM

Kesimpulan:

Nested list digunakan untuk membuat list berlevel.
