radiobutton

Pendahuluan

Seperti halnya radio button, check box juga digunakan untuk input yang berupa pilihan. Penampilannya pun hampir sama dengan radio button, namun perbedaannya adalah bahwa dalam check box memungkinkan user memilih item lebih dari satu. Tag untuk membuat check box hampir sama dengan radio button hanya saja type-nya checkbox.
