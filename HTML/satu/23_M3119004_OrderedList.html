Ordered List

Pendahuluan

Ordered list menggunakan tag <ol> dan setiap itemnya menggunakan tag <li> seperti tag <ul>

Contoh koding:

<h3>Cara <i>shutdown</i> di Windows 10</h3> <ol> <li>Buka Start</li> <li>Klik ikon Power</li> <li>Kemudian, klik <i>Shutdown</i></li> </ol>

Hasil:

Cara shutdown di Windows 10

  1. Buka Start
  2. Klik ikon Power
  3. Kemudian, klik Shutdown

Kesimpulan:

Ordered list digunakan untuk daftar yang dibuat berurutan.
