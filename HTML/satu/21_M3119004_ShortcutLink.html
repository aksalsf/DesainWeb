Shortcut Link

Pendahuluan

The accesskey attribute specifies a shortcut key to activate/focus an element.

Contoh koding:

<a href="#hal1" accesskey="v">Halaman 1</a> (Alt+V, Ctr+V)<br /> <a href="#hal2" accesskey="w">Halaman 2</a> (Alt+W, Ctr+W)<br /> <a href="#hal3" accesskey="x">Halaman 3</a> (Alt+X, Ctr+X)<br />

Hasil:

Halaman 1 (Alt+V, Ctr+V)
Halaman 2 (Alt+W, Ctr+W)
Halaman 3 (Alt+X, Ctr+X)

Kesimpulan:

Hyperlink akan aktif ketika shortcut link ditekan.
